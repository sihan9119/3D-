# 3D-粒子交互
摸鱼用gemini搓的赛博千里江山图和赛博昙花

你可以再利用Gemini或其他方式按需增减功能，但请不要以分享的形式进行盈利，禁止商用，如需转载请注明出处。

prompt：
用Three.js创建一个实时交互的3D粒子系统。
1.通过摄像头检测单手张合控制粒子群的缩放与扩散
2.提供UI面板可选择爱心/花朵/土星/烟花等模型
3.支持颜色选择器调整粒子颜色
粒子需实时响应手势变化
4.界面简洁现代，包含全屏控制按钮。
以上只是一个提示词框架，根据自己的需求修改。
千里江山图和昙花需要对细节进行尽可能详细的描述，都是我改了很多次才出来的效果，所以我没有具体的提示词
gemini生成的代码保存成html文件，在浏览器里打开使用（参考@WhisperWisp ）

有友友反应windows摄像头识别手势要等模型加载出来才能使用，可能比较慢，mac好像没有这个问题

昙花颜色rgb如下（把对应数字输入到调色盘里就行了）：
第一个颜色 r 118 g 99 b 16
第二个颜色 r 65 g 35 b 58
第三个颜色 r 106 g 82 b 107
第四个颜色 r 228 g 203 b 47
